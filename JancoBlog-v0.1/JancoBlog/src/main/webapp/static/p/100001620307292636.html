图像傅里叶变换的MATLAB实现

本文基于《MATLAB图像处理实例详解》[杨丹，赵海滨，龙哲]2013年版

图像的二维离散傅里叶变换

图像的二维离散傅立叶变换，代码如下

  1. % 读入图片
  2. I = imread("img/test1.png");
  4. % 变换之后为 complex double类型
  5. J = fft2(I);
  7. % double类型
  8. K = abs(J/256);
  10. figure;
  11. subplot(121); imshow(I); % 显示图像
  12. subplot(122); imshow(uint8(K)); % 显示频谱图

为什么要除以256 ？

首先将图片数据转化为实数类型。这一步将0-255的像素值转化为0.0-1.0范围内的实数。大多数图像处理API支持整数和实数类型的输入。如果输入是整数类型，这些API会在内部将输入转化为实数后处理，再将输出转化为整数。如果有多个处理步骤，在整数和实数之间的反复转化将导致精度损失，因此推荐在图像处理前将其转化为实数类型

imshow(uint8(K)); 为什么要将图片转化为uint8形式？

matlab中，常使用imshow()函数来显示图像，而此时的图像矩阵可能经过了某种运算。

在matlab中，为了保证精度，经过了运算的图像矩阵I其数据类型会从unit8型变成double型。

如果直接运行imshow(I)，我们会发现显示的是一个白色的图像。这是因为imshow()显示图像时对double型是认为在0~1范围内，即大于1时都是显示为白色，而imshow显示uint8型时是0 ~ 255范围。

而经过运算的范围在0-255之间的double型数据就被不正常得显示为白色图像了。

傅里叶变换后平移

  1. % 读入图片
  2. I = imread("img/test1.png");
  3. % 傅里叶变换
  4. J = fft2(I);
  5. % 类型转换
  6. K = abs(J/255);
  7. % 对原图像进行平移
  8. K = fftshift(K); 
  9. figure;
  10. subplot(121); imshow(I); title('原图像');
  11. subplot(122); imshow(uint8(K)); title('傅里叶变换平移之后的图像');

变亮后进行傅里叶变换

  1. % 读取图像
  2. R = imread("img/test2.png");
  3. % 将图片变为灰度图像
  4. I = rgb2gray(R);
  5. % 将图像变亮，因为I为uint8类型，所以不用担心超出255的范围。
  6. I1 = I * exp(1);
  7. % 进行傅里叶变换
  8. J1 = fft2(I); % 原图
  9. J2 = fft2(I1); % 变亮后
  10. % 转换
  11. K1 = abs(J1/255);% 原图
  12. K2 = abs(J2/255);% 变亮后
  13. % 图像平移
  14. K1 = fftshift(K1); % 原图
  15. K2 = fftshift(K2); % 变亮后
  16. figure;
  17. subplot(221); imshow(R); title('原图像');
  18. subplot(222); imshow(I); title('灰度图像');
  19. subplot(223); imshow(uint8(K1)); title('傅里叶变换平移之后的图像');
  20. subplot(224); imshow(uint8(K2)); title('变亮之后的图像');

旋转后进行傅里叶变换

  1. % 读取图像
  2. R = imread("img/test2.png");
  3. % 将图片变为灰度图像
  4. I = rgb2gray(R);
  5. % 将图像逆时针45度旋转
  6. I1 = imrotate(I, 45, 'bilinear');
  7. % 进行傅里叶变换
  8. J1 = fft2(I); % 原图
  9. J2 = fft2(I1); % 平移后
  10. % 转换
  11. K1 = abs(J1/255);% 原图
  12. K2 = abs(J2/255);% 平移后
  13. % 图像平移
  14. K1 = fftshift(K1); % 原图
  15. K2 = fftshift(K2); % 平移后
  16. figure;
  17. subplot(221); imshow(R); title('原图像');
  18. subplot(222); imshow(I); title('灰度图像');
  19. subplot(223); imshow(uint8(K1)); title('傅里叶变换平移之后的图像');
  20. subplot(224); imshow(uint8(K2)); title('平移之后的图像');

添加高斯噪声

  1. % 读取图像
  2. R = imread("img/test2.png");
  3. P1 = R; % 原彩色图像
  4. % 将图片变为灰度图像
  5. I = rgb2gray(R);
  6. P3 = I;
  7. % 添加高斯噪声
  8. I1 = imnoise(I, 'gaussian', 0, 0.03);
  9. P2 = imnoise(R, 'gaussian', 0, 0.03);
  10. P4 = I1;
  11. % 进行傅里叶变换
  12. J1 = fft2(I); % 原图
  13. J2 = fft2(I1); % 添加后
  14. % 转换
  15. K1 = abs(J1/255);% 原图
  16. K2 = abs(J2/255);% 添加后
  17. % 图像平移
  18. K1 = fftshift(K1); % 原图
  19. K2 = fftshift(K2); % 添加后
  20. figure;
  21. subplot(321); imshow(P1); title('原图像');
  22. subplot(322); imshow(P2); title('高斯噪声');
  23. subplot(323); imshow(P3); title('原图像');
  24. subplot(324); imshow(P4); title('高斯噪声');
  25. subplot(325); imshow(uint8(K1)); title('原图像');
  26. subplot(326); imshow(uint8(K2)); title('高斯噪声');

傅里叶反变换

  1. % 读取图像
  2. R = imread("img/test2.png");
  3. % 将图片变为灰度图像
  4. I = rgb2gray(R);
  5. % 进行傅里叶变换
  6. J = fft2(I);
  7. % 图像平移
  8. L = fftshift(J);
  9. % 平移回来
  10. M = ifftshift(L);
  11. % 傅里叶反变换
  12. M = ifft2(M);
  14. figure;
  15. subplot(121); imshow(I); title('原图');
  16. subplot(122); imshow(uint8(M)); title('反变换');
