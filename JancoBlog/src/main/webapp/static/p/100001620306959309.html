队列 — 链队

本文为作者学习b站上交版<<数据结构与算法>>, 结合清华大学出版社<<数据结构(C语言版)>> 的学习笔记, 部分内容来自于视频中的的PPT

定义

只能在队首进行删除, 队尾进行性插入的线性表
特征 : FIFO (先进先出)

从图中我们可以看出, 一个队列有一个队头和一个队尾, 入队从队尾入, 出队从队头出, 每个节点由数据域和指针域构成, 其中的每个指针域指向靠近队尾的元素

注意 : front结点的数据域是为空的, 真正有数据的的头结点是 front->next

链式队列

表示

  1. //队列
  3. typedef int QElemType;
  5. //链队列
  7. typedef struct Qnode {
  8.     QElemType data;
  9.     struct Qnode* next;
  10. }Qnode, *QueuePtr;
  12. typedef struct {
  13.     QueuePtr front;
  14.     QueuePtr rear;
  15. }LinkQueue;
  17. //代码来源 <<数据结构>> (清华大学出版社出版, 严蔚敏/吴伟民 编著)

初始化

  1. Status InitQueue(LinkQueue &L) {
  2.     L.front = (Qnode*)malloc(sizeof(Qnode*));
  3.     if (!L.front) return (OVERFLOW);
  4.     L.front->next = NULL;
  5.     L.rear = L.front;
  6.     return OK;
  7. }
  8. //代码来源 <<数据结构>> (清华大学出版社出版, 严蔚敏/吴伟民 编著)

分配头结点空间并将next域置空, 空队列的头指针和尾指针重合

入队

队尾入队, 队头出队

  1. Status EnQueue(LinkQueue& L, QElemType e) {
  2.     QueuePtr temp = (QueuePtr)malloc(sizeof(QueuePtr));
  3.     if (!temp) return (OVERFLOW);
  4.     temp->data = e;
  5.     temp->next = NULL;
  6.     L.rear->next = temp;
  7.     L.rear = temp;
  8.     return OK;
  9. }
  11. //代码来源 <<数据结构>> (清华大学出版社出版, 严蔚敏/吴伟民 编著)

开辟新节点, 将其data域赋值为e, next域置空, 将L 的尾指针指向temp, 后将其作为L的尾端

出队

队尾入队, 队头出队

  1. Status DeQueue(LinkQueue& L, QElemType& e) {
  2.     if (L.front == L.rear) return ERROR;
  3.     QueuePtr p = L.front->next;
  4.     e = p->data;
  5.     L.front->next = p->next;
  6.     if (L.rear == p) L.rear = L.front;
  7.     free(p);
  8.     return OK;
  9. }
  11. //代码来源 <<数据结构>> (清华大学出版社出版, 严蔚敏/吴伟民 编著)

由于front指向队首的下一个位置, 所以在出队的时候, 是让front的next出队

销毁

  1. Status DestoryQueue(LinkQueue& L) {
  2.     while (L.front){
  3.         L.rear = L.front->next;
  4.         free(L.front);
  5.         L.front = L.rear;
  6.     }
  7.     return OK;
  8. }
  10. //代码来源 <<数据结构>> (清华大学出版社出版, 严蔚敏/吴伟民 编著)

清空

  1. Status ClearQuqeue(LinkQueue& L) {
  2.     while (L.front != L.rear){
  3.         L.rear = L.front->next;
  4.         free(L.front);
  5.         L.front = L.rear;
  6.     }
  7.     return OK;
  8. }

清空和销毁的区别是, 清空会留下最后一个节点

因为清空的循环判定条件是front != rear

判断是否为空

  1. Status QueueEmpty(LinkQueue& L) {
  2.     return (L.front == L.rear);
  3. }

链式队列为空的条件 : 头指针和尾指针重合

返回长度

  1. int Length(LinkQueue& L) {
  2.     if (L.front == L.rear) return ERROR;
  3.     int length = 0;
  4.     LinkQueue p = L;
  5.     while (p.front->next) {
  6.         length++;
  7.         p.front = p.front->next;
  8.     }
  9.     return length;
  10. }

当临时变量p的next指向非空的时候, length++, 最后返回length

获得队头元素

  1. Status GetHead(LinkQueue& L, QElemType& e) {
  2.     e = L.front->data;
  3.     return OK;
  4. }

直接返回头指针指向节点的data域

遍历输出

  1. Status QueueTraverse(LinkQueue& L) {
  2.     if (L.front == L.rear) return ERROR;
  3.     LinkQueue p = L;
  4.     while (p.front->next) {
  5.         printf("%d", p.front->data);
  6.         p.front = p.front->next;
  7.     }
  8.     return OK;
  9. }

用临时变量p来访问队列中的元素
