线性表的顺序表示

线性表的顺序结构与实现

前提 : 

  1. #define TRUE 1 //正确
  2. #define FALSE 0 //错误
  3. #define OK 1 //执行成功
  4. #define ERROR 0 //执行有误
  5. #define INFEASIBLE -1 //不可执行
  6. #define OVERFLOW -2 //溢出
  8. typedef int Status; //函数执行的结果, 返回值为 int 类型
  10. typedef struct {
  11.     int* elem; //存储空间基地址
  12.     int length; //当前长度
  13.     int listsize; //当前分配的存储容量 以 sizeof(int) 为单位
  14. }SqList;
  16. Status InitList_Sq(SqList& L)
  17. {
  18.     //构造一个空的线性表 L
  19.     L.elem = (int*)malloc(LIST_INIT_SIZE * sizeof(int));
  20.     if (!L.elem) exit(OVERFLOW); //存储空间分配失败
  21.     L.length = 0; //空表的长度为 0
  22.     L.listsize = LIST_INIT_SIZE; // 初始的存储容量
  23.     return OK;
  24. } //InitList_Sq
  26. //代码来源 <<数据结构>> (清华大学出版社出版, 严蔚敏/吴伟民 编著)

向线性表中插入元素

  1. Status ListInsert_Sq(SqList& L, int i, int e)
  2. {
  3.     // 在顺序线性表的第 i 个位置之前插入新元素 e
  4.     // i 的合法值 为 1 <= i <= L.Lenth + 1
  5.     if (i < 1 || i > L.length + 1) return ERROR; //i 值不合法 
  6.     if (L.length >= L.listsize) { //当前的存储空间不够, 增加分配
  7.         int* newbase = (int*)realloc(L.elem, (L.length + LISTINCREMENT) * sizeof(int));
  8.         if (!newbase) exit(OVERFLOW); //存储空间分配失败
  9.         L.elem = newbase; //新基址
  10.         L.listsize += LISTINCREMENT; //增加存储容量
  11.     }
  12.     int* q = &(L.elem[i - 1]); //q 为插入位置
  13.     for (int* p = &(L.elem[L.length - 1]); p >= q; --p)
  14.         *(p + 1) = *p; //插入的部分及其后的元素右移
  15.     *q = e; // 插入 e
  16.     ++L.length;
  17.     return OK;
  18. } //ListInsert_Sq
  20. //代码来源 <<数据结构>> (清华大学出版社出版, 严蔚敏/吴伟民 编著, 这里由于不知道ElemType是啥所以改成了int, 留了个坑打算以后再填)

​ 在第 6 行, 我们比较了 L.lenth 与 L.listsize 的大小, 由于 listsize 表示的是 L 中所能存储的数据大小的上限, 它是以一个 sizeof(int) 为单位的, 所以当 L.length >= L.listsize 的时候, 我们要增加分配的空间.

​ 在第 7 行, 我们使用了一个新的指针 newbase , 来接收 L.elem 扩大后的基址 , 并且将 其再赋值给 L.elem , 让 L.elem 的内存增大.

​ 然后我们将 插入元素及其后的元素 的位置后移, 再将新元素赋值给原来那个 要插入的元素的位置 即可.

关于realloc

( 来源 : 百度百科-realloc )

首先要引入头文件 #include

语法 :

指针名=（数据类型*）realloc ( 要改变内存大小的指针名，新的大小 )

新的大小可大可小

​ 如果新的大小大于原内存大小，则新分配部分不会被初始化

​ 如果新的大小小于原内存大小，可能会导致数据丢失

功能 :

先判断当前的指针是否有足够的连续空间，如果有，扩大mem_address指向的地址，并且将mem_address返回，如果空间不够，先按照newsize指定的大小分配空间，将原有数据从头到尾拷贝到新分配的内存区域，而后释放原来mem_address所指内存区域（注意：原来指针是自动释放，不需要使用free），同时返回新分配的内存区域的首地址。即重新分配存储器块的地址。

合并两个线性表

  1. void MergeLisst_Sq(SqList& La, SqList& Lb, SqList& Lc)
  2. {
  3.     //已知线性表 La Lb 的元素是按照 值 非递减 排列 
  4.     //归并 La 与 Lb 得到新的顺序线性表, Lc 的元素也按照值 非递减排列
  5.     int* pa = La.elem;
  6.     int* pb = Lb.elem;
  7.     Lc.listsize = Lc.length = La.length + Lb.length;
  8.     int* pc = Lc.elem = (int*)malloc(Lc.length * sizeof(int));
  9.     if (!Lc.elem) exit(OVERFLOW); //空间分配失败, 终止程序
  10.     int* pa_last = La.elem + La.length - 1;
  11.     int* pb_last = Lb.elem + Lb.length - 1;
  12.     while (pa <= pa_last && pb <= pb_last) //归并
  13.     {
  14.         if (*pa < *pb)
  15.             *pc++ = *pa++;
  16.         else 
  17.             *pa++ = *pb++;
  18.     }
  19.     while (pa < pa_last) *pc++ = *pa++; // 插入a中剩余的元素
  20.     while (pb < pb_last) *pc++ = *pb++; // 插入b中剩余的元素
  21. } //MergeLisst_Sq
  23. //代码来源 <<数据结构>> (清华大学出版社出版, 严蔚敏/吴伟民 编著, 这里由于不知道ElemType是啥所以改成了int, 留了个坑打算以后再填)

归并操作 (12 — 20 行) :

最初, 我们的 pa 与 pb 都是指向了 线性表 a 和 b 的 初始位置, 原理是 每次都把 线性表 a 和 b 里面的较小的一个元素赋给 c, 然后指向 c 的指针和指向 a , b 中较小元素的指针同时右移(后移), 而指向较大的元素的指针 不变, 直到 a 或 b 中的全部元素进入 c ( 如果 a b 中元素个数相同, 则 while 语句完成之后 pa 和 pb 都是指向a 和 b的末尾, 即 pa_last 和 pb_last, 不进行 19 和 20 行 ) , 这样, 有剩余元素的那个 线性表, 里面剩余的元素肯定是比 c 中原有的都要大的, 所以直接插入即可.

比较两个线性表的大小

例1 : 设A=（a1, a2, a3 … am），B=（b1, b2, b3 … bn）为两个线性表，试写出比较A，B大小的算法。

比较原则：首先去掉A，B两个集合的最大前缀子集之后如果A，B为空，则A = B；如果 A 空 B 不空，A < B；如果 B 空 A 不空，则A > B；如果 A 和 B 均不空，则首元素大者为大。

  1. int compare(SqList A, SqList B)
  2. {
  3.     // 若 A < B 返回 -1, A > B 返回 1
  4.     int j = 0;  //控制循环
  5.     while (j < A.length && j < B.length)
  6.     {
  7.         if (A.elem[j] < B.elem[j])
  8.             return -1;
  9.         else if (A.elem[j] > B.elem[j])
  10.             return 1;
  11.         else
  12.             j++;
  13.     }
  14.     if (A.length == B.length)
  15.         return 0;
  16.     else if (A.length < B.length)
  17.         return -1;
  18.     else
  19.         return 1;
  20. }

我们将这个功能写成一个函数, 将判断的结果作为函数的返回值 : 若 A < B 返回 -1, A > B 返回 1 , 相等则返回 0 .

我们用 j 来控制循环, 在A 和 B 的最小长度的范围内 , 只要有一个元素不相等, 就返回相应的表示判断结果的值, 不然的话就继续循环.

当 j 达到了 A 或 B 的长度的时候, 表示至少有一个线性表中已经没有元素了, 此时跳出循环, 如果 A, B 的长度相等, 则A, B 中的所有的元素都相等 A = B, 返回 0; 如果 A 的长度小于 B 的长度, 则 A < B , 返回 -1 , 否则返回 1.

单个线性表的整体互换

例 2 ：设计一个算法，用尽可能少的辅助空间, 将顺序表中前m个元素和后n个元素进行整体互换。即将线性表（a1，a2…，am, b1，b2…，bn）→（b1，b2…，bn，a1，a2…，am） 要求 : 空间复杂度尽可能小

  1. void exchange(SqList& L, int m, int n)
  2. {
  3.     //线性表分成两个部分之后, 两部分倒置
  4.     int temp; // 中间变量作为 首元素 的'中转站'
  5.     for (int i = 0; i < m; i++)
  6.     {
  7.         temp = L.elem[0]; //将首元素挪到中转站
  8.         for (int j = 0; j < L.length - 1; j++)
  9.             L.elem[j] = L.elem[j + 1]; //首元素后面的元素 全部 前移 1 个单位
  10.         L.elem[L.length - 1] = temp; // 首元素 到表的末尾, 完成一次移动
  11.     }
  12. }
